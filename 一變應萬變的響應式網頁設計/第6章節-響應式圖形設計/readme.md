## 基礎篇:響應式圖形設計
* width:100% 圖片會依照父元素放大或縮小，圖片放大會有模糊問題
* max-width:100% 圖片會依照父元素縮小，最大就是原圖的大小
## 響應式圖片reset
    img{
      max-width:100%;
      height: auto;
    }
## 心法篇
* 圖片size規劃，刻意設計比較大一點的圖片
## SVG:向量圖片介紹
