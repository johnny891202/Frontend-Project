# CSS3技巧
## 背景圓弧效果
* border-radius: 10px;
* 完整圓型 border-radius: 50%;
* 如果希望只有左上角 border-radius: 50% 0 0 0;
* 在改變形狀後可以再外加 borber
## 背景漸層效果
* background:linear-gradient(上面的顏色, 下面的顏色);
* 如果要左到右, background:linear-gradient(to right, blue, red);
## 陰影效果
* box-shadow: x軸數值 ｙ軸數值 柔化距離 color;
* x軸ｙ軸數值為正時往右下, 柔化距離不能為負數
